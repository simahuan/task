# HttpDemo

#### 项目介绍
Okhttp最新版本3.10网络请求的极致封装，包含get和POST的同步异步，字符串和json传参，
文件上传和下载，cookie和缓存的设置。是复习和更新OKhttp的利器，是学习和进阶Retrofit的基础文档。
